# Siemens SI — Hierarchical Sales Forecasting

**NOVA IMS · Siemens Advanta Consulting Case Study · 2026**

## Objective
Develop a consistent hierarchical forecasting approach for Siemens Smart Infrastructure sales KPIs (New Orders / Revenue), ensuring that GCK-level forecasts are coherent when aggregated up through the organizational hierarchy (Subsegment → Segment → BU → SI total).

## Data

- **Training set**: `Case2_data_extract_share_1.xlsx` (sheet `training`) — ~4,200 rows, columns: `Anon Period`, `TGL Business Unit`, `TGL Business Segment`, `TGL Business Subsegment`, `Orders cons. (anon)`, `Revenue cons. (anon)`
- **Validation set**: sheet `validation` — ~700 rows, same structure
- **Hierarchy map**: sheet `BUs` — mapping BU → Segment → Subsegment (GCK)
- **Market data**: `Case2_market_data_share_3.xlsx` — 78 macro indicators (GDP, Industrial Production, Inflation, etc.) for 9 countries, 180 monthly periods. Sheet 2 maps `Anon Period` → real calendar dates.

---

## Modelling Approach Suggestion 

1. **Seasonal naïve baseline** — repeats the value from the same month of the prior year, per GCK
2. **LightGBM with external features** — lag features + macro indicators + calendar features
3. **MinT reconciliation** — projects base forecasts onto the hierarchically coherent subspace

Evaluation metric: **RMSE** (quantitative submission to SAC)
